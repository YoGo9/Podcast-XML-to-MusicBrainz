
# Podcast XML to MusicBrainz

WIP - Made to help import podcast episodes into MusicBrainz.

I've used AI tools to help me build this.

Try it here: https://podbrainz.com/

Please feel free to modify and send pull requests!


## Deployment Instructions

### Deploy with Vercel (Recommended)

1. Fork this repository to your GitHub account
2. Sign up for a [Vercel account](https://vercel.com) if you don't already have one
3. Connect your GitHub account to Vercel
4. Click "Import Project" in Vercel dashboard
5. Select your forked repository
6. Keep all default settings and click "Deploy"
7. Once deployed, Vercel will provide you with a URL for your application

### Local Development

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/podcast-to-musicbrainz.git
   cd podcast-to-musicbrainz
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the local development server:
   ```
   vercel dev
   ```

4. Open your browser to `http://localhost:3000`
